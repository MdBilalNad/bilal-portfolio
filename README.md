# Mohammed Bilal Nadeem - Portfolio Website

Interactive portfolio website for Mohammed Bilal Nadeem, an AI/ML Engineer.

## Features

- 🎨 Sleek, modern design with dark theme
- 📊 Interactive skill bars with animations
- 🔄 Smooth scrolling and transitions
- 📱 Fully responsive for all devices
- 🎯 Measurable skill metrics
- 📄 Resume download functionality
- 🖼️ Animated profile section
- 🔗 Social media integration

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (ES6)
- AOS (Animate On Scroll)
- Font Awesome Icons
